📍 Location
(0, 0)
The base tile is at (0, 0). Farming actions (till, plant, water, harvest) are no-ops on the base — they do nothing and waste a tick. Use is_at_base() to check your position.
💡
Return to base when energy or tank runs low. The drone recharges automatically — no function call needed. At max upgrades: 360 energy, 150 tank.
// Go home to recharge, then come back if (get_energy() < 20 || get_tank() < 15) { while (!is_at_base()) { move_west(); move_north(); } wait(20); // charge up }

Every PLANTED or GROWING cell loses 2 water/tick. When water hits 0, growth pauses — the crop doesn't die, it just stops progressing. Water can go above 50 via water() up to the cell max of 100.
🍄
Mushroom is special — it only grows when water_level > 40. You must water proactively to keep the level high, not reactively when it runs out.
if (get_water_level() < 30 && get_tank() >= 15) water(); // costs 15 tank, gives cell +50 water

🌿 Crop Suppliers
Four crop types are locked by default. Unlock them by purchasing a Supplier in the 🚁 Extra panel. Once unlocked, you can buy seeds and plant that crop. Suppliers persist until you reset the game.
💡
Unlock Corn first — it has the best gold/tick of the non-special crops (0.133). Mushroom needs a smart watering algorithm to beat it (0.150 g/tick when water is maintained). Coffee gives the highest absolute profit: +15 gold per harvest.
⚡ Quick Start Algorithm
A complete working program. Sweeps the field east→west, farms every cell, returns to base when resources are low.
int main() { while (true) { // Recharge if low on energy or tank if (get_energy() < 30 || get_tank() < 30) { while (get_x() > 0) move_west(); while (get_y() > 0) move_north(); wait(15); continue; } // Buy seeds if needed if (get_seeds(CropType::POTATO) < 5) buy_seeds(CropType::POTATO, 10); // Farm the current cell CellState s = get_cell_state(); if (s == CellState::EMPTY) till(); if (s == CellState::TILLED) plant(CropType::POTATO); if (s == CellState::READY) harvest(); if (get_water_level() < 25) water(); // Move: sweep east, then step south and go back west if (get_x() < get_max_x() - 1) { move_east(); continue; } if (get_y() < get_max_y() - 1) move_south(); while (get_x() > 0) move_west(); } }
